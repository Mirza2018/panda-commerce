## **Getting Started**

### **Installation**

To get started with this project, clone the repository and install the dependencies:

```bash
https://github.com/Mirza2018/panda-eommerce
cd panda-eommerce
npm install
```

### **Running the App**
```bash
cd panda-eommerce
npm run dev
```
